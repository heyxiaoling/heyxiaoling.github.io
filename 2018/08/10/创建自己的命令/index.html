创建自己的命令工具

Veröffentlicht am 2018-08-10 | in Basic

创建自己的命令工具

1. 创建文件夹

mkdir heyxiaoyuandoudou //为了不重复

2. 初始化

npm init -y

创建bin文件夹并创建文件index.js,内容如下（也可不创建bin文件夹）

console.log('hello');

修改package.json

"bin": {
    "heyxiaoyuandoudou": "./bin/index.js"
},

login ，publish

npm login //输入用户名密码邮箱
npm publish

可以在C:\Users\用户\AppData\Roaming\npm\node_modules下面找到heyxiaoyuandoudou文件夹，其实是系统变量存储在这里

如何使用

npm install -g heyxiaoyuandoudou
heyxiaoyuandoudou  //hello

如何更新

修改package.json的version,(顺便修改./bin/index.js)

"version": "1.0.1",
console.log('hello 更新版本');

//提交到npm
npm publish //命令不变 //提示+ heyxiaoyuandoudou@1.0.0

//更新使用
npm updata heyxiaoyuandoudou -g

//使用
heyxiaoyuandoudou
//hello 更新版本

使用辅助模块

  1. commander：命令行开发工具
  2. chalk：命令行样式风格控制器
  3. minimist：命令行参数解析器

commander使用

npm i commander / yarn add commander
//引入
const program = require('commander');

.parse(process.argv)
//解析用户在命令行中输入的参数内容
//process.argv：一个包含了命令行参数的数组

//默认选项
//-h, --help    打印输出使用（帮助）信息
.version()
version(str: string, flags?: string): Command
//设置工具版本信息
//同时会自动添加一个选项
-V, --version    //查看当前版本
//可以通过 flags 修改默认的 -V, --version 选项
version('1.0.0', '-v, --version')

.usage()
usage(str: string): Command
//设置程序的usage说明

.arguments()
arguments(desc: string): Command
//设置参数

搭配使用

const program = require('commander');
const fs = require('fs');
const chalk = require('chalk');
program
.command('create <project_app>')
.description('创建项目')
.action(function(project_app) {

let projectPath = __dirname + "/" + project_app;

if (fs.existsSync(projectPath)) {
    console.log( chalk.red(`项目：${project_app} 已经存在了` ) );
    return;
}

console.log( chalk.yellow(`即将创建项目：${project_app}` ) );
console.log('projectPath',projectPath);
fs.mkdirSync(projectPath);
console.log( chalk.green(`项目目录创建成功` ) );

fs.writeFileSync(projectPath + '/index.html');
console.log( chalk.green(`项目首页创建成功` ) );

fs.mkdirSync(projectPath + '/css');
console.log( chalk.green(`项目css文件夹创建成功` ) );

fs.mkdirSync(projectPath + '/js');
console.log( chalk.green(`项目js文件夹创建成功` ) );

console.log( chalk.green(`项目构建完成` ) );

});


program.parse( process.argv );
